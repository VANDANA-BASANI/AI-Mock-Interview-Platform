# AI-Powered Mock Interview Platform

## Overview

AI-Powered Mock Interview Platform is a web application that helps users prepare for job interviews through realistic mock interview sessions. Users can choose a job role, company, difficulty level, and number of questions, then answer technical, behavioral, and situational interview questions while receiving performance feedback.

## Features

- Role-based interview practice
- Technical, Behavioral, and Situational questions
- Difficulty selection (Easy, Medium, Hard)
- Interactive interview experience
- Real-time progress tracking
- Performance scoring system
- Question-wise feedback
- Overall interview evaluation
- Responsive and modern UI

## Technologies Used

- React.js
- TypeScript
- Vite
- Tailwind CSS
- Radix UI
- Lucide React Icons

## How It Works

1. Select a job role and company.
2. Choose difficulty level and number of questions.
3. Answer interview questions one by one.
4. Receive feedback for each response.
5. View overall performance score and improvement suggestions.
